# NEON SPRAWL — Colony Protocol

A cyberpunk colony sim in a single HTML file. Vanilla JS + canvas, zero dependencies.

**Play:** open `index.html` in any modern browser (desktop + touch supported), or via GitHub Pages once enabled.

**Goal:** keep your runners fed, rested, and alive. Salvage scrap, wall in, get power online, then build the Extraction Beacon and keep it powered for one full day.

**Controls:** click/drag select · R drafts · right-click commands (move / focus-fire) · WASD + wheel camera · space pauses · 1/2/3 speed · ? in-game field manual.

**Dev:** edit `index.html`, then verify with `node test/run.js` (headless test suite — no browser needed). See `CLAUDE.md` for architecture and workflow rules.
